# 4.区块

# 4.1区块的基本

区块引用是在段落开头使用>符号

区块引用 回车换行到下一行自动加上> 即使不加>也会一直处于一个区块内

两个回车即空一行结束一个区块

# 4.2区块的嵌套

一个>表示最外层

两个>>表示第一层嵌套

三个>第二层

# 4.3区块中的列表

	return a+b
